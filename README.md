# Artificial Neural Network from Scratch (Python)

This project is a custom implementation of a **Multilayer Perceptron (MLP)** built entirely in Python using only the `NumPy` library. 

[cite_start]Unlike implementations using high-level frameworks (like TensorFlow or PyTorch), this project features a manual development of the **Backpropagation** algorithm and gradient calculations, demonstrating a deep understanding of AI fundamentals and linear algebra. [cite: 6, 60, 122]

## 🚀 Key Features
* [cite_start]**Modular Architecture**: Independent classes for Neurons, Layers, and the main Neural Network. [cite: 61, 67, 121, 122]
* [cite_start]**Manual Mathematics**: Implementation of the Sigmoid activation function and weight adjustments via gradient descent. [cite: 122]
* [cite_start]**Data Versatility**: Configured to process and classify multiple datasets including Iris, Mushrooms, and Penguins. [cite: 65, 72, 123]
